# Tech Stack

## 💻 Frontend
- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Styling**: TailwindCSS
- **UI Components**: Radix UI (via shadcn/ui)
- **State Management**: Zustand (Client State), React Query (Server State/Caching)
- **Icons**: Lucide React
- **Forms & Validation**: React Hook Form, Zod

## ⚙️ Backend
- **Architecture**: Next.js Server Actions & API Routes
- **Authentication**: NextAuth v5 (Auth.js beta)
- **Real-time**: Socket.IO (Server & Client)

## 🗄️ Database
- **Database**: PostgreSQL
- **ORM**: Prisma

## 🛠️ Tooling & Validation
- **Linting & Formatting**: Biome / ESLint / Prettier
- **Package Manager**: pnpm (implied by pnpm-lock.yaml)
