# Project Brief: ZoZo Booking

## 🎯 Overview
ZoZo Booking is a comprehensive dining and order management system designed for restaurants. It allows customers to browse menus, place orders via QR codes, and enables restaurant staff to manage tables, orders, in real-time.

## 👥 Target Audience
- **Customers**: Diners at the restaurant looking for a seamless, contactless ordering experience.
- **Restaurant Staff/Admin**: Managers and waitstaff who need to track table statuses, manage incoming orders, and update menus.

## ✨ Core Features
- **Digital Menu & Ordering**: Customers can view the menu, customize options, and place orders directly from their devices.
- **Table Management**: Real-time tracking of table statuses (Available, Occupied, etc.).
- **Real-time Order Sync**: Orders are instantly updated for both customers and staff using WebSocket connections.
- **Restaurant Admin Dashboard**: Interface for managing categories, menu items, item options, and viewing active orders.
- **Authentication**: Role-based access control (Admin, Staff, Customer).

## 🚀 Future Enhancements (Roadmap)
- Push Notifications
- Analytics & Reporting Dashboard
- Payment Integration
- Multi-language Support (i18n)
- Reviews & Ratings
